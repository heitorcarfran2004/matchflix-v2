# **App Name**: MatchFlix Elite

## Core Features:

- Hero Section with Video: Display a background video with a headline, subheadline, and subscription plans.
- Creators Carousel: Show a horizontal carousel of creators using provided images.
- Platform Benefits: List platform benefits with icons and clear descriptions.
- Premium Creators Section: Display premium creators with video previews and overlay CTAs. The overlay CTA invites users to 'See Plans'.
- FAQ Section: Implement an FAQ section with accordion-style questions and answers.
- Overlay Script: Implement a JS script that displays an overlay upon video completion with a CTA.

## Style Guidelines:

- Primary color: Deep, muted red (#C0392B) to convey a premium and exclusive feel.
- Background color: Dark charcoal (#222222) to provide a clean and elegant backdrop.  Visibly the same hue as the primary, but desaturated.
- Accent color: Soft orange (#D35400) for calls to action and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look.
- Mobile-first approach, ensuring a fully responsive design that adapts to different screen sizes.
- Use rounded corners and subtle shadow effects to enhance visual appeal and create a premium feel.
